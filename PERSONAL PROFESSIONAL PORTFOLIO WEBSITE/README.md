
# Personal Portfolio Website

## Overview
This is a responsive personal portfolio website developed using HTML, CSS, Bootstrap, and JavaScript. The website showcases personal information, skills, projects, and contact details.

## Features

- Responsive Design
- Modern UI
- Hero Section
- About Me Section
- Skills Showcase
- Projects Gallery
- Contact Form
- Social Media Links
- Bootstrap Integration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Font Awesome

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click `index.html`.

OR

Use VS Code Live Server:

1. Install Live Server Extension.
2. Right click on `index.html`.
3. Select **Open with Live Server**.
